# 🍕 PizzaSaaS - Sistema Ordini Online per Pizzerie Sistema multi-tenant per creare siti di ordinazione online per pizzerie in pochi minuti. ## 🌐 File del Sistema ### **1. index.html** - Sito principale - Pizzeria demo funzionante - Sistema di ordinazione completo - Invito alla registrazione - Carrello compatto e responsive ### **2. registrazione.html** - Pagina registrazione - Form a 3 step guidati - Inserimento menu pizze personalizzato - Selezione colori tema - Anteprima in tempo reale ### **3. landing.html** - Sito vetrina - Presentazione del servizio - Features e benefici - Demo pizzerie - Call-to-action per registrazione ## 🚀 Come Usare 1. **Cliente visita** `landing.html` o `index.html` 2. **Clicca "Registrati"** per creare il suo sito 3. **Compila il form** in 3 step: - Step 1: Informazioni pizzeria - Step 2: Menu pizze - Step 3: Personalizzazione colori 4. **Sito creato automaticamente** con reindirizzamento ## 🎯 Funzionalità ### Per il Cliente Finale: - ✅ Sistema ordinazione completo - ✅ Carrello persistente - ✅ Menu personalizzato - ✅ Design responsive - ✅ Multi-tenant (ogni pizzeria ha il suo sito) ### Per il Pizzaiolo: - ✅ Setup in 2 minuti - ✅ Personalizzazione completa - ✅ Nessuna competenza tecnica richiesta - ✅ Gestione menu intuitiva ## 📱 URL del Sistema
